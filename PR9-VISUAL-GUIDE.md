## Color Coding System / سیستم رنگ‌بندی

```
🟢 Green (Excellent/Success)
   • Health Score: 80-100
   • Status: Everything working well
   • Action: Continue monitoring

🔵 Blue (Good/Info)
   • Health Score: 60-79
   • Status: Performing well
   • Action: Minor optimizations

🟡 Orange/Yellow (Warning)
   • Health Score: 40-59
   • Status: Needs attention
   • Action: Review recommendations

🔴 Red (Critical/Error)
   • Health Score: 0-39
   • Status: Immediate action required
   • Action: Fix critical issues
```

## Usage Scenarios / سناریوهای استفاده

### Scenario 1: Morning Check (30 seconds)
```
1. Open Atlas Dashboard
2. Check health score
3. Read priority alerts
4. Done!
```

### Scenario 2: Weekly Review (15 minutes)
```
1. Layer 1: Review overall health
2. Layer 2: Check bottlenecks
3. Layer 3: Read recommendations
4. Take action on high-priority items
```

### Scenario 3: Before Campaign Launch
```
1. Layer 1: Ensure site health is good
2. Layer 2: Fix any critical bottlenecks
3. Layer 4: Simulate expected impact
4. Layer 5: Adjust monitoring settings
5. Launch campaign
```
